# 🚗 Nigerian Car Market Dashboard (Excel)

## 📌 Objective
To analyze the Nigerian car market using Excel and build an interactive dashboard to track key vehicle metrics and trends.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Cleaning

## 📂 Dataset
The dataset includes:
- Car brands and models
- Price (₦)
- Mileage (km)
- Horsepower (hp)
- Engine size (cc)
- Transmission type
- Car color
- Location data

## 🔍 Key Features
- KPI cards (Price, Mileage, Horsepower, Engine Size)
- Top 10 car models analysis
- Transmission preference (Automatic vs Manual)
- Regional distribution (map visualization)
- Car color popularity
- Interactive filters (Brand New, Foreign Used, Nigerian Used)

## 📊 Insights
- Automatic cars dominate the market
- Popular colors include Silver and White
- Price varies based on condition and specifications
- Regional trends highlight demand differences
